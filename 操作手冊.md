## 一、系統簡介

本系統透過 LINE Bot 提供 GPS 打卡、請假/加班/補打卡申請、多階簽核、每日報表等功能。

**四種角色權限：**

| 角色 | Rich Menu | 打卡 | 申請 | 簽核 | 查詢範圍 |
|------|-----------|------|------|------|----------|
| 一般員工 | 6 格 | ✅ | ✅ | ❌ | 僅個人 |
| 簽核人員 | 8 格 | ✅ | ✅ | ✅（指定員工） | 指定員工 |
| 經理 | 8 格 | ✅ | ✅ | ✅ | 全體員工 |
| 老闆 | 4 格 | ❌ | ❌ | ❌ | 全體（唯讀） |

## 二、一般員工操作

### 2.1 Rich Menu 功能表（6 格）

LINE 聊天室下方固定顯示 6 個按鈕（3×2）：

| 上班 | 請假 | 下班 |
|------|------|------|
| 加班 | 補打卡 | 查詢 |

點選按鈕即可觸發對應功能。

### 2.2 GPS 打卡

**上班打卡：**
1. 點選「**上班**」
2. 系統提示「請分享您的位置」
3. 點選左下角 **📍 分享位置** 按鈕
4. 系統回傳打卡結果（時間、位置、是否遲到、GPS 範圍）

**下班打卡：**
1. 點選「**下班**」
2. 同上步驟分享位置
3. 系統回傳打卡結果（時間、工時、是否未滿 9 小時）

**卡控規則：**

| 情況 | 系統回應 |
|------|----------|
| 當天已上班打卡 | ⚠️ 今天已上班打卡 |
| 尚未上班打卡就打下班 | ⚠️ 尚未上班打卡 |
| 當天已下班打卡 | ⚠️ 今天已下班打卡 |
| 上下班均完成 | 今日已完成打卡 ✅ |

**工時提醒：**
- 正常上班時段（8:00–17:30）工時未滿 9 小時 → ⚠️ 警告
- 下班時間超過 18:30 → ⚠️ 提醒記得報加班

### 2.3 查詢個人紀錄

點選「**查詢**」或輸入「查詢」，系統回傳兩則訊息：

**第一則（Flex Message）：今日概況**
- 🔵 上班時間與位置
- 🔴 下班時間與位置
- 📊 總工時 / 淨工時 / 是否未滿 9h

**第二則（文字訊息）：當月考勤明細**
- ⚠️ 遲到紀錄（日期、時間、晚幾分、已請假備註）
- 🏖 請假明細（日期區間、假別、時數、累計）
- 🕐 加班明細（日期、時間、時數、累計）

### 2.4 請假申請

**操作流程：**
1. 點選「**請假**」或輸入「請假」
2. 選擇假別：特休 / 事假 / 病假 / 公假 / 外出 / 其他
3. 📅 選擇開始日期時間
4. 📅 選擇結束日期時間
5. 系統自動驗證（結束 ≥ 開始）
6. 輸入請假原因
7. 送出 → 進入簽核流程

**請假時數計算規則：**
- 跨天每日最多 8 小時
- 午休 12:00–13:00 自動扣除 1 小時
- 最小時數為 1 小時

**注意：** 若遲到時間已被核准請假覆蓋，遲到紀錄保留但備註「已請假」。

### 2.5 加班申請

**操作流程：**
1. 點選「**加班**」或輸入「加班」
2. 📅 選擇開始日期時間
3. 📅 選擇結束日期時間
4. 系統自動驗證（僅允許 17:30–23:00 時段）
5. 輸入加班原因
6. 送出 → 進入簽核流程

### 2.6 補打卡申請

**操作流程：**
1. 點選「**補打卡**」或輸入「補打卡」
2. 選擇補上班卡 / 補下班卡
3. 📅 選擇補打卡日期時間
4. 系統自動驗證：
   - ❌ 不可補未來時間
   - ❌ 限 3 天內
   - ❌ 當天已有同類型打卡 → 不可補
5. 輸入原因
6. 送出 → 核准後自動寫入打卡記錄

### 2.7 其他指令

| 輸入文字 | 功能 |
|----------|------|
| `我的ID` 或 `my id` | 顯示 LINE User ID（綁定用） |
| `取消` | 取消目前操作流程 |

## 三、簽核人員操作

### 3.1 Rich Menu 功能表（8 格）

簽核人員的 Rich Menu 比一般員工多 2 個查詢按鈕（4×2）：

| 上班 | 請假 | 補打卡 | 下班 |
|------|------|--------|------|
| 加班 | 查詢 | **查詢當天考勤** | **查詢當月考勤** |

### 3.2 簽核操作

**收到簽核通知時：**
- 請假/加班/補打卡申請會以 Flex Message 推送給簽核人
- 訊息內含：申請人、日期時間、原因、**核准** 與 **駁回** 按鈕
- 點選「**駁回**」時可輸入駁回原因

**批次操作（Quick Reply 選單）：**
- 「**核准全部**」— 一鍵核准所有待審核的請假單
- 「**駁回全部**」— 一鍵駁回所有待審核的請假單
- 「**加班核准全部**」— 一鍵核准所有待審核的加班單
- 「**加班駁回全部**」— 一鍵駁回所有待審核的加班單

**簽核層級說明：**
- 最多三階簽核（L1 → L2 → L3）
- 每階通過後，系統自動通知下一階簽核人
- 全數通過 → 通知申請人「已核准」
- 任一駁回 → 通知申請人「被駁回」
- 若該階無簽核人 → 自動跳過（直接核准）

**簽核人員僅能看到/處理自己負責員工的申請。**

### 3.3 查詢被簽核人員

**查詢當天考勤**（左下 3）：
- 被簽核人員當天狀態：
  - ⚠️ 遲到人員與時間
  - ❌ 曠職人員（未打卡且未請假）
  - 🏖 請假中人員與假別
  - 📍 GPS 超出範圍人員

**查詢當月考勤**（右下 3）：
- 被簽核人員 1 號～當天：
  - ⚠️ 遲到紀錄（日期/時間/晚幾分/是否已請假）
  - 🏖 請假明細（日期區間/假別/時數/累計）
  - 🕐 加班明細（日期/時間/時數/累計）

## 四、經理操作

### 4.1 與簽核人員的差異

經理使用相同的 8 格 Rich Menu，但權限更大：

| 功能 | 簽核人員 | 經理 |
|------|----------|------|
| 打卡/申請 | ✅ | ✅ |
| 簽核 | 僅指定員工 | 全體員工 |
| 查詢當天考勤 | 僅指定員工 | 全體員工 |
| 查詢當月考勤 | 僅指定員工 | 全體員工 |
| 批次核准/駁回 | ✅ | ✅ |

### 4.2 後台管理

經理可登入網頁後台進行以下管理：

| 頁面 | 功能 |
|------|------|
| 📊 儀表板 | 今日出勤統計、請假概況、最近打卡 |
| 📋 打卡記錄 | 篩選/查詢/匯出打卡記錄、遲到統計 |
| 👥 員工管理 | 新增/編輯/離職、LINE 綁定、設定簽核人 |
| 🏖 請假管理 | 篩選/核准/駁回/刪除請假、匯出 Excel |
| 🕐 加班管理 | 篩選/核准/駁回/刪除加班、匯出 Excel |
| 📝 補打卡 | 審核補打卡申請 |
| 📦 資料彙整 | 一鍵匯出四合一 Excel（出勤彙總+打卡+請假+加班） |
| 💵 薪資發送 | 編輯薪資單（文字+圖片）、排程推送 |
| ⚙️ 系統設定 | 上下班時間、GPS 座標/範圍、國定假日、日報設定 |

## 五、老闆操作

### 5.1 Rich Menu 功能表（4 格）

老闆不參與打卡與簽核，使用專屬 4 格選單（2×2）：

| 公司今日考勤 | 本月請假累計 |
|--------------|--------------|
| 本月遲到累計 | 本月加班累計 |

### 5.2 四個查詢功能

**1. 公司今日考勤** — 當天全公司考勤快照：
- ⚠️ 遲到人員（時間、遲到分鐘、是否已請假）
- ❌ 曠職人員（未打卡且未請假）
- 🏖 請假中人員（假別）
- 📍 GPS 超出範圍人員
- 自動排除假日

**2. 本月請假累計** — 1 號～月底：
- 每位請假人員的明細（日期區間、假別、時數）
- 個人累計時數 + 全公司合計

**3. 本月遲到累計** — 1 號～當天：
- 每位遲到人員的明細（日期、時間、晚幾分）
- 若該次遲到已請假覆蓋 → 備註「已請假」
- 個人累計次數 + 全公司合計
- 自動排除假日

**4. 本月加班累計** — 1 號～當天：
- 每位加班人員的明細（日期、時間、時數）
- 個人累計時數 + 全公司合計

## 六、考勤規則說明

### 6.1 遲到判斷

- 上班時間 + 緩衝分鐘（預設 8:00 + 30 分鐘 = 8:30）
- 超過 8:30 打卡 → 標示遲到及晚到分鐘數
- **假日與國定假日不計遲到**

### 6.2 曠職判斷

當天同時滿足以下條件 → 曠職：
- 無上班打卡記錄
- 無請假（已核准）
- 無補打卡（已核准）

### 6.3 工時計算

- **總工時** = 下班時間 − 上班時間
- **淨工時** = 總工時 − 午休（若跨越 12:00–13:00 扣 1 小時）
- **正常工時** = 8:00–17:30 內的工時（17:30 後屬加班），未滿 9 小時標記 ⚠️

### 6.4 國定假日

- 系統設定頁面可維護國定假日清單
- 假日上班打卡不計遲到
- 假日不上班不計曠職

### 6.5 請假覆蓋遲到

- 若遲到打卡的時間區間被已核准的請假覆蓋
- 遲到紀錄保留（次數照計），但備註「已請假」
- 例：8:35 打卡遲到，但已請 8:30–9:00 的假 → 顯示「8:35 晚 5 分 已請假」

## 七、常見問題

### Q1：忘記打卡怎麼辦？
A：使用「補打卡」功能，限 3 天內申請，需經簽核。

### Q2：GPS 超出範圍仍可打卡嗎？
A：可以，系統會標示「超出範圍」但仍記錄打卡。管理人員可在後台查看。

### Q3：請假被駁回怎麼辦？
A：系統會通知駁回原因，可重新申請。

### Q4：為什麼查詢不到今天的請假人員？
A：請假需在日期範圍內（開始日期 ≤ 今天 ≤ 結束日期）且狀態為「已核准」才會顯示。

### Q5：國定假日打卡算遲到嗎？
A：不算。系統會自動排除假日與國定假日的遲到判斷。

### Q6：Rich Menu 沒出現怎麼辦？
A：強制關閉 LINE App 再重新開啟。如仍無效，請聯絡管理員重新設定。

### Q7：可以補打卡未來時間嗎？
A：不行，系統會阻擋未來時間的補打卡。

### Q8：請假時數是怎麼算的？
A：每天最多 8 小時，午休 12:00–13:00 扣 1 小時，跨天分段計算。

### Q9：簽核人員和經理有什麼差別？
A：簽核人員只能看到/處理「被指定為簽核人」的員工；經理可查看/處理全體員工。

### Q10：怎麼看我的 LINE User ID？
A：在 LINE 聊天室輸入「我的ID」，系統會回傳你的 User ID，供管理員綁定用。
